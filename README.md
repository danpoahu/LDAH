## üöÄ Quick Start Guide

### Step 1: Create App Icons (REQUIRED)
You need two icon files:
- `icon-192.png` (192x192 pixels)
- `icon-512.png` (512x512 pixels)

**Easiest method:**
1. Go to https://www.pwabuilder.com/imageGenerator
2. Upload your logo
3. Download generated icons
4. Rename to icon-192.png and icon-512.png

See **ICON_CREATION_GUIDE.md** for detailed instructions.

### Step 2: Upload All Files
Upload these 10 files to your website root directory:
- All 6 HTML files
- manifest.json
- service-worker.js
- icon-192.png (you create this)
- icon-512.png (you create this)

### Step 3: Test Installation
**On iPhone:**
1. Open website in Safari
2. Tap Share ‚Üí Add to Home Screen
3. See LDAH icon on home screen!

**On Android:**
1. Open website in Chrome
2. Tap Menu ‚Üí Add to Home screen
3. See LDAH icon on home screen!

- Manage volunteer applications

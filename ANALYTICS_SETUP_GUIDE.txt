FIREBASE ANALYTICS COLLECTION:
-------------------------------

All tracking data is stored in Firebase Firestore in a collection called 'analytics'.

Each event includes:
- eventType (page_view, event_signup, volunteer_application_submitted)
- timestamp
- page location
- Additional data specific to the event type

TROUBLESHOOTING:
----------------

**No data showing?**
- Make sure you deployed all the updated files
- Click "Refresh Data" button
- Check that Firebase is working (test by submitting a signup)

**Numbers seem low?**
- Remember: tracking only starts from when you deploy these updates
- Historical data before this update is not tracked
- Check your date range settings

**Need help?**
- All data is in Firebase under the 'analytics' collection
- You can always export raw data from Firebase console
